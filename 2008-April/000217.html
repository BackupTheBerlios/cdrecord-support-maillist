There is no function called "fmtab" in cdrecord.

Line 2058 in cdrecord.c contains a call to fillbytes()

Given the fact, that there have recently been some bug reports on optimizer
bugs in gcc-4.2.3 that cause fillbytes() to be compiled incorrectly, I would 
recommend to either use an older GCC or the latest GCC which seems to be 
gcc-4.3.
