You did not include a suficcient amount of information to give you more help, 
however you are not running cdrecord but a variant from redhat with inofficial
modifications.

So far all these modified cdrtools versions have been identified to be 
defective.

Please use a recent cdrtools version from

ftp://ftp.berlios.de/pub/cdrecord/alpha/

and make sure to install cdrecord suid root.
