After quite a lot of trial and error, I've come up with a combination of versions which will reliably write discs piped from mkisofs to cdrecord under Windows, without needing a full cygwin shell (just the DLLs). Since this took a bit of work, I thought I'd post it here so it's in the archives. I'll state complete version numbers where possible so others can recreate this situation.

I don't know if this is one of Mr Schilling's preferred original versions of cdrecord, but it works and it's extremely useful so it's worth mentioning.

So there you go, you -can- pipe between mkisofs and cdrecord in windows without a cyg shell.
