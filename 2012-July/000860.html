I have now done a number of tests with cdrskin 
(http://scdbackup.webframe.org/cdrskin_eng.html) and can confirm that it 
can burn multi session discs with this media and drive. Example commandline:

========================================================================

cdrskin -v -fs=8M -multi -stream_recording=on -eject -dev=/dev/sr0 
'/mnt/Storage_1/Temporary Storage Folder/Processing 
Folder/Burning/disc_image.iso'

========================================================================

stream_recording kills off the bluray defect management.
