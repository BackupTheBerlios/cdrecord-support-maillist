smake is automake enhanced, as it has been compiled with Cygwin, it calls 
uname(2) from cygwin and sets up defaults for Cygwin.

smake retrieves many things from the OS:

MAKEFLAGS  =    -pr 
MAKE_ARCH  =    i386 
MAKE_BRAND  =   
MAKE_DOMAIN  =  
MAKE_FLAGS  =   -pr 
MAKE_HOST  =    opt 
MAKE_HWSERIAL  =        448596686 
MAKE_ISALIST  = amd64 pentium_pro+mmx pentium_pro pentium+mmx pentium i486 i386 i86 
MAKE_LEVEL  =   1 
MAKE_MACH  =    i86pc 
MAKE_MODEL  =   i86pc 
MAKE_M_ARCH  =  i86pc 
MAKE_NAME  =    smake 
MAKE_OS  =      sunos 
MAKE_OSDEFS  =  -D__SVR4 
MAKE_OSREL  =   5.11 
MAKE_OSVERSION  =       snv_105 
MAKE_SHELL_FLAG  =      -ce 
MAKE_SHELL_IFLAG  =     -c 
MAKE_VERSION  = 1.2.1 
NUMBER_SIGN  =  # 
SHELL  =        /bin/sh 

This all has been setup by smake. On Cygwin, smake will in addition give you 
information about the underlying WIN-DOS.

Compiling for mingw using cygwin tools is called "cross compilation" and in 
such a case, you need to overwrite all relevant make macros from the command 
line to smake.
