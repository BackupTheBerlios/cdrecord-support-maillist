"Phil_Rhodes at rocketmail.com" <phil_rhodes at rocketmail.com> wrote:

> Hi,
>
> I'm writing some code around cdrecord which needs to recognise which of the devices listed by -scanbus are writers. About the best approach I've got is to try and match the manufacturer's description strings to a regex like  DVD.?R[-W_ ]. This is frankly a very nasty approach, but is about the best I can come up with, and bug fixes should be reasonably easy.
>
> There are, however, a lot of blu-ray writers out there which will also write DVDs. I'd like to collect as many description strings as I can for these writers so I can start to construct a filter which will recognise blu-ray writers. If anyone's got one, please let me know what -scanbus says about it!
>

The correct way to find this is to call

cdrecord -v -checkkdrive

If you are using a writer made after ~ 2001, you will get a list of 
supported media.
