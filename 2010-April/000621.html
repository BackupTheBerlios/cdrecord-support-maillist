I see what he means. Tis an English thing. In the man page of my
older cdrecord (so this may have been changed) it says about
-r:

"This is like the -R option, but file ownership and modes are set
 to more useful values."

"like the -R option" can be confusing because it is not clear in
what way something is like something else. In the extreme case,
all options are "like" each other because they are all command
options. LOL.

Better to say something such as "The same as the -R option, except
file ownership and modes are set to more useful values."
