And to get better efficiency one could use cdrtools' readcd. See 
the EXAMPLES section of man page:

        To  read the complete media from a CD-ROM writing the data to
        the file cdimage.raw:

            readcd dev=2,0 f=cdimage.raw

(where the dev values are an example, of course)
