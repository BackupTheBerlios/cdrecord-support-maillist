Mark Knecht <markknecht at gmail.com> wrote:

> Hi,
>    Is cdrecord the right tool to write a new BIOS image to a CD? I'd
> like to update my BIOS. Inside of BIOS it says it will read the new
> BIOS image off of a CD but I'm having trouble finding an example in
> man cdrecord to just write a single file.
>
>    The new BIOS image file is named RampageII-ASUS-Extreme-2101.ROM.

The BIOS updates I've seen so far are ISO images that contain a FreeDOS 
installation inside an emulated Floppy disk.


You can verify this using isoinfo.


In such a case just write the raw image to the CD using cdrecord
