If in doubt, not Wikipedia but the original documentation from CDRWIN counts.

A manual that includes the official docs can be found here:

	ftp://209.113.146.144/public_ftp/cdrwin.pdf


As the CUE file looks OK, what writer do you use? If this is a Lite-ON, be 
prepared that the time stamps will count backwards because of a firmware 
bug.....


If you like to make sure that not your drive is the cause, write a CD in 
-raw96r mode.
