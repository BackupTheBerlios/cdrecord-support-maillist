> There is no need to reboot.
>
>
> If there is no /dev/fbk0, the postinstall did not work correctly. Run
>
> drvconfig
> devlinks
>
> this should create it.
>
> If this is no ufs, then you need to mount the image by hand later using:
>
> mount -F something  /dev/fbk0 /mnt
>
> Theassociation between /dev/fbk0 and the file is removed after the last close.
