> QUESTION: I have an audio CD with a multimedia track on it that works
> when you play the CD in Windows. Is there a way to duplicate this CD
> in its entirety? If you need info off the CD or about the drive let me

This is in the documentation:

Those CDs are usually milti-session CDs

1)	extract the audio tracks using cdda2wav

2)	write the audio tracks using cdrecord -sao -multi .....

3)	call cdrecord -toc on the original disk and then
	call readcd sectors=.... f=xxx.iso with the right sector
	range for the data rack in the 2nd session

4)	write the data track using cdrecord -tao -multi ....
	to append it to the audio tracks
