Mark Knecht <markknecht at gmail.com> wrote:

> Looking at the examples in man cdrecord I don't see an example for
> cloning a DVD movie. Is cdrecord an appropriate tool for doing this
> and if so what would a reasonable set of commands be to accomplish
> that task?

If the DVD is completely unencrypted, you may read the data via readcd and then 
write it.

If it is encrypted, you need to call DVDcopy to get an unencrypted copy on your 
local filesystem and then run mkisofs in DVD movie mode (see man page).
