Mark Knecht <markknecht at gmail.com> wrote:

> How would I determine if a specific CD has hidden tracks so that can
> tell if this new version is working correctly? Does TOC list the track
> with (I guess) a negative number or something?

What people call "hidden Track" is data in the pregap for track #1.
This is present when the start for track #1 is past sector 0 (2:00 seconds)
and when the data in the related sectors is non-null.
