  Jorge,


It seems TSSTcorp stands for Toshiba Samsung Storage Technology, so
Samsung firmware probably should work given it is the same model
number.

There is a fellow who said he has updated his TSSTcorp SH-S182M:

http://forum.videohelp.com/threads/269868-possible-to-upgrade-DVD-writer-firmware-in-Linux-(e-g-using-wine-)

under Linux using a Window's flasher (does the poster mean the Dangerous
Brother's or Samsung flasher?) under Wine, the Linux Windows emulator:

http://www.winehq.org/

That said, if the flasher runs under a DOS command prompt, and you
have a means to boot that, then I would go that route because it
might be safer.

Searching Google for 

DOS boot image

shows a lot of places where the needed files can be obtained
quickly.

Do let us know how it goes. One reason I prefer NEC and Optiarc
drives is that there is a Linux flash program (in binary form)
for them (although I wish it would be released open source)
as well as modified NEC firmware:

http://liggydee.cdfreaks.com/page/en/
