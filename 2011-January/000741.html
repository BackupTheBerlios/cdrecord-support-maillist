  Daniel,

I haven't seen such a problem and maybe you will get a good answer
from somebody else, but just in case you don't, I have found that
in cases like this it can help to try an alternative burn application
to try to isolate the problem and see if there might be a possible
media/firmware/hardware problem.

You can try the dvd+rw-tools:

http://fy.chalmers.se/~appro/linux/DVD+RW/

http://fy.chalmers.se/~appro/linux/DVD+RW/tools/?M=D

and see if they work OK. e.g.,

growisofs -dvd-compat -Z /dev/dvd=file.iso

If growisofs stalls fixating like cdrecord, you're probably looking
at a media, kernel, firmware or hardware issue. However, if the
problem still happens, then cdrecord might be more of a suspect.
