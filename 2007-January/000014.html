Hi,

i have compiled the current (18th Dec) .a23 on Linux (Debian Sarge  
3.1) with the normal gcc just by a "make" in the src directory.  To  
avoid conflicts with the cdrecord of Debian i did not use the make  
install but copied the binaries to places where i like to have them.
I have burned many DVDs on a NEC 3540A with it.
Same compile worked without any hassle on mac os x 10.4 both for i386  
and ppc.  Just had to install Apple's XCODE. Did not find a way to  
crosscompile on i386 for ppc though or to make universal binaries.

My suggestion - if possible get the source and compile it, it is  
quite simple.

Georg
