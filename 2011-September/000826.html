Hi,
   This report is for the Gentoo stable version of cdrtools-3.00. I
note there is a newer testing version 3.01_alpha05 which I have not
tested.

   I was looking for an example a couple of days ago to duplicate a
data CD with a bunch of pictures on it for my wife. Reading the man
page for cdrecord I found the text quoted below.

   I think the first example should likely not have the -clone option
included to cdrecord? Removing that created a correct copy as best I
could tell

Cheers,
Mark


[QUOTE]

   Replicating a simple CD-ROM/DVD-ROM/BD-ROM
       To copy a simple disk, first read the master using:

           readcd dev=1,0 f=somefile

       Then write the disk using:

           cdrecord dev=2,0 -raw96r -clone -v somefile


   Replicating a CD-ROM in clone mode
       To copy a CD in clone mode, first read the master CD using:

           readcd dev=1,0 -clone f=somefile

       or (in case the CD contains many sectors that are unreadable by
intention) by calling:

           readcd dev=1,0 -clone -nocorr f=somefile

       This will create the files somefile and somefile.toc.  Then
write the CD using:

           cdrecord dev=2,0 -raw96r -clone -v somefile

[/QUOTE]
